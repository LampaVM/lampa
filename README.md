Lampa plugins:
https://cub.red/plugin/collections - это расширение для медиаприложения Lampa, которое добавляет в интерфейс отдельный раздел «Коллекции» с тематическими подборками фильмов и сериалов.

https://skaztv.online/store.js - это магазин плагинов (Store) для медиаплеера Lampa.Он служит альтернативным каталогом плагинов, через который можно в удобном виде устанавливать дополнительные «Пиратские плагины»
